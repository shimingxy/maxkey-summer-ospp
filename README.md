# summer-ospp-开源之夏

summer-ospp 是“开源之夏”（Open Source Promotion Plan, OSPP）的官方域名简称，由中国科学院软件研究所主办、华为等联合支持的‌面向全球高校学生的暑期开源开发活动‌ 。‌‌

*核心目标‌*：鼓励在校大学生参与真实开源项目开发，在导师指导下贡献代码，获取奖金与结项证书，助力开源生态建设 。
